# Stay Event - Plataforma de Gestión de Entradas 🎟️

### 🚀 Descripción
Proyecto desarrollado para el 9no ciclo de Ingeniería de Software. "Stay Event" es una solución web para la compra y venta de entradas a eventos, enfocada en la autogestión y seguridad mediante códigos QR.

### 🛠️ Stack Tecnológico
* **Frontend:** React.js + Tailwind CSS
* **Backend:** Node.js + Express
* **Base de Datos:** MySQL
* **Herramientas:** Visual Studio Code, Git, GitHub

### 📁 Estructura del Proyecto
* `client/`: Interfaz de usuario y lógica de frontend.
* `server/`: API REST, controladores y modelos de base de datos.
